Frequently Asked Questions

Is my video uploaded to a server?

For the real version using FFmpeg.wasm, the processing happens locally in your browser! In this demo, no file actually leaves your computer.

What is the max file size?

We recommend files under 2GB for browser performance. For anything larger (>3GB), desktop software is usually more stable.

Will I lose video quality?

If you choose "Balanced" or "High Quality", the visual difference is negligible. "Smaller Size" might show slight artifacts but saves huge space.

Can I use this on mobile?

Yes! The interface is fully responsive. However, processing large 4K videos on older phones might be slow.
